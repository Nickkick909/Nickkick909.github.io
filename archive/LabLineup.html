Lab Lineup is part of my senior design project at the University of South Carolina.
It is a web application written in Python, HTML, and CSS with the Django framework.
This is an app to help students request a TA/professor to help them in a lab environment.
Students can submit their request online, and the TA/professor can see the queue and accept the next request.
This is a group project with collaboration using GitHub.
